# Trae-Agents-Prompt

## Repository Introduction

Trae-Agents-Prompt is a meticulously designed repository of AI Agent prompt templates, dedicated to providing developers with high-quality, reusable intelligent agent workflow templates. This repository collects specialized Agent instruction sets across multiple domains to help you quickly build and customize AI assistants suitable for specific tasks.

## Repository Contents Overview

This repository currently contains the following professional Agent prompt templates:

- **CodeAnalyzer.md** - Code architecture topology analysis expert, focused on analyzing code structure characteristics and module boundary relationships from a pure engineering perspective, generating objective, language-independent architecture documentation
- **Bugfixer.md** - Intelligent debugging assistant that quickly identifies error types in code, locates root causes, restores developer intent, and provides precise repair solutions
- **Requirement Refiner.md** - Requirement refinement expert that systematically breaks down vague user requirements, narrows scope, and transforms them into executable minimum viable iteration plans

## Agent Template Structure Standards

Each Agent prompt template follows a unified structural design to ensure consistency and usability:

- **Role Definition**: Clearly defines the Agent's professional identity, scope of responsibility, and core capabilities
- **Core Principles**: Basic guidelines and work constraints that the Agent must follow when performing tasks
- **Workflow**: Detailed work steps, methodologies, and decision paths
- **Output Specifications**: Clear output format requirements, quality standards, and examples
- **Interaction Rules**: Specific norms and considerations for interacting with users or systems

## Usage Guide

### Basic Usage Method

1. Browse the Agent template files in the repository and select the template that best suits your needs
2. Copy the template content to a platform or tool that supports AI conversations
3. Adjust template parameters and context information according to the specific scenario
4. Provide task-related specific inputs to the AI to initiate the intelligent agent workflow

### Advanced Customization Suggestions

- Adjust role definitions and professional terminology in templates according to specific domain needs
- Provide more contextual information to the Agent to obtain more accurate output
- Combine the advantages of multiple Agent templates to build complex workflow solutions

## Contribution Guidelines

We greatly welcome community contributions, including new Agent templates and improvements to existing templates:

1. **Fork** this repository to your personal account
2. **Create a new branch** (recommended to use `Agent-<Name>` or `Improve-<TemplateName>` format)
3. **Write or modify** Agent prompt templates, ensuring they comply with the repository's structural standards
4. **Submit a Pull Request** with a description of your changes and purpose
5. Wait for review and feedback from the repository maintainers

## Naming Conventions

To maintain the cleanliness and consistency of the repository, please follow these naming conventions:

- New Agent template files use **PascalCase** naming (e.g., `CodeAnalyzer.md`)
- File names should be concise and directly reflect the Agent's main function or professional domain
- Multilingual versions are distinguished using file suffixes (e.g., `README.en.md` for English version)

## Language Support

- Main documents and Agent templates are written in Chinese to ensure ease of use for domestic users
- English versions of documents and templates are distinguished with the `.en.md` suffix for international users
- Community contributions of translations in other languages are welcome
