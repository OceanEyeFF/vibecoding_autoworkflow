# Harness Skill Catalog

`docs/harness/Skills/catalog/` 按 `Codex` 实际消费方式组织 Harness skills。

当前入口：

- [supervisor.md](./supervisor.md)
- [repo.md](./repo.md)
- [worktrack.md](./worktrack.md)

组织原则：

- `supervisor`：顶层 Harness round supervisor
- `repo`：`RepoScope` 下直接可调用的 skills
- `worktrack`：`WorktrackScope` 下直接可调用的 skills

这里记录的是 skill catalog，不再额外抽象成单独的 function 目录。
