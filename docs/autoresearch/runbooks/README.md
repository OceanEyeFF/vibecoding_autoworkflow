# Autoresearch Runbooks

`docs/autoresearch/runbooks/` 只保存 autoresearch 与其运行时维护相关的 repo-local runbook。

当前入口：

1. [autoresearch-minimal-loop.md](./autoresearch-minimal-loop.md)
2. [tmp-exrepo-maintenance.md](./tmp-exrepo-maintenance.md)

这里适合放：

- autoresearch 最小运行闭环
- TMP exrepo 维护动作
- 与 research runner 紧邻的运维步骤

这里不适合放：

- deploy backend 总览
- backend skill mount 维护
- canonical 模块边界正文
