## 使用规则

1. `WORKBOARD` 只记录已准入任务，不记录纯想法。
2. `SPRINT` 中出现的 ID 必须来自本文件。
3. 状态改为 `done` 之前，必须先在 `CHANGELOG.md` 记录结果。
