# operations（运行手册）

> 本目录存放 repo-local runbook 与执行模板。

## prompt-templates

- `simple-subagent-workflow.md`：中小任务的轻量执行模板。
- `strict-subagent-workflow.md`：跨模块/高风险任务的严格执行模板。
- `execution-contract-template.md`：Step 1 固定中间产物模板。
- `harness-contract-template.json`：Harness 结构化合同模板（含 scope/gates/risk/status）。
- `review-loop-code-review.md`：commit/PR 代码优先审查与修复闭环模板。
- `task-planning-contract.md`：需求/问题清单转可执行任务合同（可路由、可并行、可验证）。
- `task-list-subagent-workflow.md`：任务文件包含多个任务时的专用 SubAgent Workflow。
