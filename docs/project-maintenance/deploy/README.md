# Deploy Runbooks

`docs/project-maintenance/deploy/` 只保存当前仓库的 repo-local deploy runbook，当前主要覆盖 deploy 与 deploy maintenance。

当前入口：

1. [deploy-runbook.md](./deploy-runbook.md)
2. [skill-deployment-maintenance.md](./skill-deployment-maintenance.md)

这里适合放：

- deploy backend 总览
- repo-local / global deploy 维护动作
- `verify`、`deploy`、`--prune` 的运维节奏

这里不适合放：

- autoresearch 主流程
- research runner 操作说明
- canonical skill 真相正文
